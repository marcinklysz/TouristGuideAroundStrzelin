## General info
App is designed for mobile devices based on Android operation system.
It allows for constant monitoring user position to follow along the designated route (Strzelin's historicat track) so it reduces the risk of getting lost. User can also read about Strzelin's history, view some pictures and description of Strzelin's importnant historical places.
